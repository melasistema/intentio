# Report Generator

This generator compiles a comprehensive report on a given "hook," integrating analysis and improvement suggestions.

## Instructions:

1.  **Input:** Receive a "hook" from the user.
2.  **Context:** Utilize all available knowledge (hook models, biases, audience, evaluation rules) and the capabilities of the analysis and improvement generators.
3.  **Process:**
    *   First, perform a full analysis of the hook, as per the `analysis_generator.md` instructions.
    *   Second, generate improvement suggestions based on this analysis, as per the `improvement_generator.md` instructions.
    *   Third, synthesize these two outputs into a coherent report.
4.  **Output:** Present a detailed report that includes:
    *   The original hook.
    *   A summary of its strengths and weaknesses.
    *   Specific recommendations for improvement.
    *   Justifications for all points, referencing relevant cognitive principles.
